# Site de l'association "A Levis Danse Country"

`data/chores.yml`: la base de données de toutes les chorégraphies

`content`/<niveau>-<année>`.md`: les danses du cours <niveau> de
l'année <année>. Exemple: `content/debutant-2526.md`: les danses du
cours niveau débutant en 2025-2026
